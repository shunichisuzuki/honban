■機能
　・太陽系に関する動画を視聴することができる。
　・神話に関する動画を視聴することができる。
　・太陽系の惑星と神話に関する動画を視聴して獲得したメダルの
　　取得状況を確認することができる。

■機能詳細
　・動画はYoutube。
　・動画は全部を順番に視聴しないと、次の動画を見ることができない。
